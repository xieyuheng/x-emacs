# x-sidebar 架构

基于 dired 的简易侧边栏文件浏览器，支持快速浏览目录和预览文件。

## 概览

- **入口**: `C-s C-w` (`M-x x-sidebar`)
- **宽度**: 30 列（可通过 `x-sidebar--width` 自定义）
- **布局**: 强制双窗口——左侧 sidebar，右侧主窗口

## 数据结构

### 状态 alist: `x-sidebar--state`

```elisp
((frame1 . buffer1) (frame2 . buffer2) ...)
```

- key: frame 对象（`selected-frame`）
- value: 该 frame 的 sidebar dired buffer
- 多 frame 安全：每个 frame 有独立条目，通过 `x-sidebar--state-entry` 按需创建

窗口不存储，始终通过扫描 buffer 名称实时查找。

### 窗口发现: `x-sidebar--find-window`

扫描当前 frame 上所有窗口，找到 buffer 名以 `*x-sidebar*` 开头的窗口。返回该窗口或 nil。

## 入口: `x-sidebar`

```
sidebar 已存在 → select-window 切换到 sidebar
  |
  or
  |
dired-noselect          → 创建/配置 sidebar buffer
delete-other-windows    → 清理 frame 为单窗口
split-window ... 'left  → 左侧分屏出 sidebar，右侧保留原窗口
select-window sidebar   → 聚焦 sidebar
preview                 → 更新主窗口内容
```

## 导航

| 键 | 功能 | 实现 |
|----|------|------|
| `RET` / `l` / `<right>` | 进入目录或打开文件 | `x-sidebar--ret` |
| `h` / `<left>` | 返回父目录 | `x-sidebar--up-directory` → `x-sidebar--enter-dir` |
| `j` / `<down>` | 下一行 | `dired-next-line` |
| `k` / `<up>` | 上一行 | `dired-previous-line` |
| `C-s C-w` | 切换到主窗口 | `x-sidebar--switch-to-main` |

### `x-sidebar--ret`

- **光标在目录上**: 调用 `x-sidebar--enter-dir`
- **光标在文件上**: `select-window` 到主窗口、`find-file` 打开文件、切回 sidebar

### `x-sidebar--enter-dir`

1. 展开目录路径为绝对路径
2. 查找该目录的已有 dired buffer（`dired-buffers-for-dir`）或创建新的（`dired-noselect`）
3. 调用 `x-sidebar--setup` 配置 buffer（键绑定、隐藏详情、post-command-hook）
4. 通过 `x-sidebar--set-buffer` 存储为当前 sidebar buffer
5. `set-window-buffer` 在 sidebar 窗口中显示
6. 触发预览

### `x-sidebar--up-directory`

从 `default-directory` 计算父目录，通过 `x-sidebar--enter-dir` 进入，然后通过 `dired-goto-file` 跳转到原来的子目录条目。

## 预览: `x-sidebar--preview`

由 sidebar buffer 中的 `post-command-hook` 触发。

```
dired-get-file-for-visit → 获取光标处文件路径
与上次预览相同则跳过     → x-sidebar--last-preview
目录 → dired-noselect
文件 → find-file-noselect
set-window-buffer 更新主窗口
```

主窗口由 `x-sidebar--main-window` 查找或创建。

## 主窗口: `x-sidebar--main-window`

```elisp
当前 frame 上已有非 sidebar 窗口 → 返回之
  |
  or (无其他窗口时)
  |
创建新窗口:
  优先使用 sidebar 光标处的文件/目录
  兜底使用 other-buffer
向右分屏，保留 sidebar 宽度
```

## 键绑定汇总

| 键 | 命令 |
|----|------|
| `C-s C-w` | `x-sidebar`（打开/切换）或 `x-sidebar--switch-to-main`（在 sidebar 内） |
| `RET` / `l` / `<right>` | `x-sidebar--ret` |
| `h` / `<left>` | `x-sidebar--up-directory` |
| `j` / `<down>` | `dired-next-line` |
| `k` / `<up>` | `dired-previous-line` |
